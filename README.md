### 1. Clone the `GithubGraphExtractor` repository

```bash
git clone https://github.com/garvitxx18/GithubGraphExtractor.git
cd gitGraphRetriever
cd client
```

### 2. Install package dependencies

```bash
npm install
npm run dev
```

### 3. TO Start the backend server



```bash
cd gitGraphRetriever
cd server
npm install
nodemon index.js
```
